하위 요소 생성: “>” 사용
/* 예: */
header>ul>li

동급 요소 생성: “+” 사용
/* 예: */
section>article>h2+p

반복 태그 생성: “*” 사용
/* 예: */
ul>li*5

CSS class 와 id 설정: “.” 와 “#”사용
/* 예: */
ul#menu>li.item*3

Grouping (그룹화) : “( )” 사용
.container>(header>nav>ul>li*5>a)+(#content>section)+footer

넘버링(숫자를 순서대로 나열): “$”사용
/* 예: */
ul.list>li.item$*5>{$}
